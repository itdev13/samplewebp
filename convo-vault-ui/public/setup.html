üìñ What is ConvoVault?

ConvoVault is a powerful conversation management tool for your CRM. It helps you:

  • Search & Filter Conversations: Advanced filtering with 8+ parameters
  • Export Messages: Download all conversation data as CSV for backup or analysis
  • Import Conversations: Bulk import conversations from CSV/Excel files (up to 5MB or 2,000 contacts)
  • API Access: Complete REST API for custom integrations

üöÄ Installation

Step 1: Install the App

Step 2: Connect Your Sub-Account

Select the sub-account where you want to install ConvoVault. You'll be redirected to authorize the connection.

Step 3: Access ConvoVault

After installation, find "Convo Vault" in your sub-account's left sidebar menu.

Step 4: Start Using

Click on ConvoVault to open the dashboard and start managing your conversations!

üìò How to Use

Search Conversations

  1. Go to the Conversations tab
  2. Set your filters (date range, status, message type, etc.)
  3. Click Search to view results
  4. Use Export to download as CSV

Export Messages

  1. Go to the Messages & Export tab
  2. Optionally filter by conversation ID
  3. Click Search to view messages
  4. Click Export All to download complete data

Import Conversations

  1. Go to the Import tab
  2. Click Download Template to get the CSV format
  3. Fill in your data (contactId/email/phone, message, etc.)
  4. Upload the file (max 5MB or 2,000 contacts)
  5. Track import status in the history section

Use the API

  1. Click the üìö API Docs button in the header
  2. Your personal API token will be displayed
  3. Copy the token and use it in your applications
  4. All API endpoints are documented with curl examples

‚ùì FAQs

How much does it cost?

ConvoVault offers a 7-day free trial. After the trial, subscription plans are $19/month. Cancel anytime with no long-term commitment.

What file formats are supported for import?

We support CSV, XLSX, and XLS files up to 5MB in size or 2,000 contacts per import.

Can I export all my conversations at once?

Yes! Use the bulk export feature to download all conversations or messages in batches.

Is my data secure?

Absolutely! We use industry-standard encryption, secure token-based authentication, and never store your conversation data permanently.

Can I use the API for custom integrations?

Yes! Access the API documentation from within the app to get your personal API token and integration examples.
