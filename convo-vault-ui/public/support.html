ðŸ”§ Common Issues & Solutions

App not loading in my sub-account

Solution: Make sure you've completed the OAuth installation. Try reconnecting your sub-account from the marketplace.

Export returns no results

Solution: Check your filters - try expanding the date range (max 1 month) or removing optional filters.

Date range exceeds limit

Solution: Exports are limited to 1 month date range. Select a shorter period and run multiple exports if needed.

Download link expired

Solution: Download links are valid for 7 days. Go to the Exports tab and click "Regenerate Link" to get a new download URL.

Payment failed

Solution: Check that your payment method is valid. Contact support if the issue persists.

ðŸ“š Feature Documentation

Export Filters

Export Messages supports multiple filter parameters:

  • Date Range - Filter by start and end date (max 1 month)
  • Channel - Filter by SMS, Email, WhatsApp, etc.
  • Status - Filter by conversation status
  • Direction - Filter by inbound or outbound messages
  • Contact ID - Export for a specific contact

Pricing

Free to install - you only pay when you export:

  • Conversations - $0.025 per conversation (1 credit)
  • SMS / WhatsApp - $0.025 per message (1 credit)
  • Email - $0.075 per email (3 credits)

Export Formats

Supported export formats:

  • CSV - Comma-separated values, compatible with Excel and Google Sheets
  • JSON - JavaScript Object Notation, ideal for developers and data processing
