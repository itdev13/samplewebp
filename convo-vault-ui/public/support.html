ðŸ”§ Common Issues & Solutions

App not loading in my sub-account

Solution: Make sure you've completed the OAuth installation. Try reconnecting your sub-account from the marketplace.

Import failed - file format error

Solution: Download the CSV template from the Import tab and ensure your file matches the format. Required fields: contactId or email/phone, and message.

Export returns no results

Solution: Check your filters - try expanding the date range or removing optional filters. Ensure the conversation ID is correct.

API authentication failing

Solution: Your session token may have expired. Refresh the app to get a new token, or use the /api/auth/refresh endpoint.

File upload size limit exceeded

Solution: Maximum file size is 5MB or 2,000 contacts per import. Split your CSV into smaller files if needed.

Advanced Filtering

ConvoVault supports 8+ filter parameters:

  • status - Filter by conversation status (active, archived)
  • lastMessageType - Filter by message type (SMS, Email, WhatsApp)
  • lastMessageDirection - Filter by direction (inbound, outbound)
  • limit - Number of results (20, 50, 100)
  • conversationId - Get specific conversation

CSV Import Format

Required columns:

  • contactId OR (email/phone)
  • message - The message text

Optional columns:

  • locationId - Sub-account ID (uses current if not provided)
  • Name - Contact details
