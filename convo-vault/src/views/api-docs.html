Authentication

POST /api/auth/refresh Protected

Refresh your session token to get a new one (expires in 1 hour).

Headers:

Header Value
Authorization Bearer {your_session_token}

Example Request:

curl -X POST https://convoapi.vaultsuite.store/api/auth/refresh \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN"

Conversations

GET /api/conversations/download Protected

Download conversations with advanced filtering and cursor-based pagination.

Query Parameters:

Parameter Type Required Description
locationId string Required Sub-account location ID
limit number Optional Number of results (default: 20, max: 100)
conversationId string Optional Filter by specific conversation ID
lastMessageType string Optional SMS, Email, WhatsApp, Call, etc.
lastMessageDirection string Optional inbound or outbound
status string Optional Conversation status filter
startAfterId string Optional Cursor for pagination
sortBy string Optional Sort criteria

Example Request:

curl -X GET "https://convoapi.vaultsuite.store/api/conversations/download?locationId=loc456&limit=20&lastMessageType=SMS" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN"
GET /api/conversations/search Protected

Search conversations with filters. Supports both general search and specific conversation lookup.

Query Parameters:

Parameter Type Required Description
locationId string Required Sub-account location ID
conversationId string Optional Get specific conversation by ID
limit number Optional Number of results

Example Request:

curl -X GET "https://convoapi.vaultsuite.store/api/conversations/search?locationId=loc456&conversationId=conv123" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN"
GET /api/conversations/:conversationId Protected

Path Parameters:

Parameter Type Description

Query Parameters:

Parameter Type Required Description

Example Request:

curl -X GET "https://convoapi.vaultsuite.store/api/conversations/conv123?locationId=loc456" \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN"

Import

POST /api/import/upload Protected

Upload CSV/Excel file to import conversations. Supports creating contacts and conversations (max 5MB or 2,000 contacts).

Form Data:

Field Type Required Description
file file Required CSV or Excel file (.csv, .xlsx, .xls)
locationId string Required Default location ID for rows without locationId

CSV Columns:

Column Required Description
locationId Optional Override default locationId per row
contactId Optional If not provided, will create contact
name Optional Contact full name
email Optional* Contact email (required if no contactId)
phone Optional* Contact phone with country code (e.g., +1234567890)
companyName Optional Contact company name
address1 Optional Street address
city Optional City
state Optional State/Province
postalCode Optional Postal/ZIP code
country Optional Country code
website Optional Website URL
timezone Optional Timezone (e.g., America/New_York)
tags Optional Tags separated by semicolons (e.g., vip;customer)

Example Request:

curl -X POST https://convoapi.vaultsuite.store/api/import/upload \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN" \
  -F "file=@conversations.csv" \
  -F "locationId=loc456"

Response:

{
  "success": true,
  "message": "Import started - processing in background",
  "data": {
    "jobId": "job123",
    "totalRows": 100,
    "status": "processing"
  }
}
GET /api/import/template Protected

Example Request:

curl -X GET https://convoapi.vaultsuite.store/api/import/template \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN" \

Support

POST /api/support/ticket Protected

Submit a support ticket with optional image attachments (max 5 images, 5MB total).

Form Data:

Field Type Required Description
email string Required Your email address
subject string Required Ticket subject
message string Required Support request details
name string Optional Your name
locationId string Optional Related location ID
userId string Optional Your user ID
images file[] Optional Up to 5 images (JPEG, PNG, GIF, WebP)

Example Request:

curl -X POST https://convoapi.vaultsuite.store/api/support/ticket \
  -H "Authorization: Bearer YOUR_SESSION_TOKEN" \
  -F "email=user@example.com" \
  -F "subject=Help with Import" \
  -F "message=I'm having trouble importing my CSV file" \
  -F "name=John Doe" \
  -F "locationId=loc456" \
  -F "images=@screenshot1.png" \
  -F "images=@screenshot2.png"

Response:

{
  "success": true,
  "message": "Support ticket submitted successfully. We will get back to you soon!"
}

Error Codes

Status Code Description
200 Success
400 Bad Request - Invalid parameters or missing required fields
401 Unauthorized - Invalid, missing, or expired token
403 Forbidden - No access to this resource
404 Not Found - Resource doesn't exist
422 Unprocessable Entity - Validation failed or duplicate resource
500 Internal Server Error - Something went wrong on our end
